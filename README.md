# Star Wars Blog - Reading List

Proyecto desarrollado con **React.js** y **Context API (useReducer)** que consume la API de [swapi.tech](https://www.swapi.tech/) para mostrar información detallada sobre el universo de Star Wars.

## 🚀 Características

- **Navegación Dinámica:** Listado de personajes, planetas y vehículos con scroll horizontal.
- **Detalles en Tiempo Real:** Vista detallada de cada elemento con datos técnicos y descripciones.
- **Sistema de Favoritos:** Agregar y eliminar elementos de una lista global accesible desde el Navbar.
- **Diseño Responsivo:** Estética "Dark Tech" inspirada en la franquicia, optimizada para diferentes dispositivos.

## 🛠️ Tecnologías Usadas

- **Frontend:** React, React Router, Bootstrap.
- **Estado Global:** Context API con el patrón `useReducer` (Global Store).
- **API:** SWAPI.tech.
- **Imágenes:** Star Wars Visual Guide.
